## 레벨 1 - 기록

### [🖋 우아한테크코스 한 달 생활기](level1.md)

---

- 잘 써야겠다는 욕심 버리기
- 같은 팀 크루 5~6명의 글에 피드백 남기기
- 더이상 남길 피드백 없을 시 200자 내외 최종 리뷰 남기고 approve
- 최종 피드백 완료시 글 작성자는 리뷰 요청 보내기

## 레벨 2 - 성장
